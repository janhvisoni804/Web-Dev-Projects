    Contribute in four steps

    Read the full guide in CONTRIBUTING.md.

    1. 01

      Fork & branch

      Fork the repo, create a branch named after your project, e.g. add-to-do-web-app.

    2. 02

      Create your folder

      Under Projects/, in Title Case with real spaces — like To Do Web App.

    3. 03

      Add the required files

      README.md, project.json and index.html. No build step, ever.

    4. 04

      Open a pull request

      Use the New Project template. Once merged, the showcase rebuilds itself.
