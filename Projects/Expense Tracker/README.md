# Expense Tracker

A premium, production-grade Black and Gold daily expense tracker featuring custom SVG charts, analytics viewports, and local storage data persistence. Built entirely on static client-side web technologies.

## Run it
Open `index.html` in any modern web browser.

## Features
- **Premium Black & Gold Theme**: Gorgeous design language optimized with soft gradients, transitions, and clear highlights.
- **Dynamic Dual Modes**: Toggle between Workspace Light Mode and Dark Mode.
- **Complete Expense Control (CRUD)**: Easily add, edit, and delete daily spending. Add transaction title, amount, category, and date.
- **Expense Categorization**: Categorize transactions (Food, Travel, Shopping, Bills, Entertainment, Others).
- **Interactive SVG Analytics**: 
  - *Monthly Chart*: Visualizes your monthly spending ratios.
  - *Donut Chart*: Dynamic vector rendering mapping category percentages.
- **State Persistence**: Draft transactions auto-save inside your local browser database (`localStorage`).
- **Responsive Layout**: Designed for mobile, tablet, and desktop dashboards.

## What I learned
- Developing custom SVG data graphs programmatically using trigonometric paths.
- Re-architecting color schemes around CSS Custom variables for light/dark interfaces.
- Manipulating localStorage indices with filter scopes.
