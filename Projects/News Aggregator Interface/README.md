# News Aggregator Interface

A web app to **browse categorized news cards, trending topics, search articles, and bookmark favorites**.

## 🚀 Features
- Categorized news cards (Technology, Sports, Business, Entertainment)
- Trending topics section
- Search bar for filtering news
- Bookmark button to save articles
