# Netflix Clone

A Netflix-inspired streaming UI with a cinematic hero banner, dynamic multi-row content rails, hover-scale cards, and a detail modal — built with vanilla HTML/CSS/JS.

## Run

Double-click `index.html` to open in your browser. No server required.

## Features

- Static hero billboard with featured title
- Horizontal-scroll content rows (Trending Now, New Releases, etc.)
- Hover-scale cards with match ratings
- Detail modal with metadata
- Fully responsive layout
