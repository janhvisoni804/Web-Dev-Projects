# LinkedIn Clone

A polished LinkedIn desktop clone built with vanilla HTML, CSS, and JavaScript.

## Features

- **Profile Banner**: Editable hero card with background, avatar, name, headline, location, company, and connection count
- **Edit Modal**: Floating workspace to modify profile info with instant preview and localStorage save
- **Post Feed**: Create posts with emoji tags, prepend to feed, like and comment on posts
- **People You May Know**: Recommendation deck with connect simulation and real-time counter
- **Print Resume**: Export/Print profile as a clean A4 CV template
- **Responsive**: 3-column desktop layout stacks vertically on mobile

## How to Run

Open `index.html` in any modern browser. No build tools or dependencies required.
