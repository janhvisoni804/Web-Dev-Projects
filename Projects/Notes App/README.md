# Notes App

A clean, keyboard-friendly notes app written in plain HTML, CSS and JavaScript.

## Features

- Create, edit and delete notes — changes save instantly to `localStorage`.
- Live URL detection: any `http(s)://` link inside a note becomes clickable.
- Tag chips parsed from `#hashtags` in the body, with one-click filtering.
- Search across titles, bodies and tags.
- Fully responsive, works offline, no dependencies.

## Run it

Open `index.html` in any modern browser. That's it.

## What it shows

- DOM updates without a framework.
- `localStorage` as a tiny database.
- Simple text parsing with regex for URLs and tags.
- Accessible keyboard interactions (Enter to save, Esc to close).
