# Friendship Letter

An animated envelope opening experience to send personalized friendship letters.  
Includes custom names, messages, colors, music, stickers, and shareable URLs.

## Features
- Animated envelope flap opening.
- Customizable names and messages via URL parameters.
- Stickers and background colors for personalization.
- Background music when the letter opens.
- Shareable link generation.
