# Weather Forecast Dashboard

A responsive weather dashboard that displays current weather conditions and a 5-day forecast using real-time weather API data.

## Features

- Search weather by city name
- Current location weather using Geolocation API
- Current temperature, wind speed and weather conditions
- 5-day weather forecast
- Responsive dashboard layout
- No API key required
- Works in modern browsers

## Run It

Open `index.html` in any modern web browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API

## What I Learned

- Working with REST APIs
- Fetching and displaying JSON data
- Handling asynchronous JavaScript with Fetch API
- Building responsive dashboard layouts
