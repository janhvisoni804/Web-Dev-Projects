# Converter Game

A dark-themed number system converter that transforms values between Binary, Decimal, Octal, and Hexadecimal instantly.

## How to Use

1. Select the input format.
2. Enter a valid value in the chosen format.
3. View the converted results in all other number systems instantly.
4. Use **Clear** to reset the input and outputs.

## Supported conversions

- Binary ↔ Decimal
- Decimal ↔ Hexadecimal
- Octal conversions
- Any format to all others automatically

## Files

- `index.html` — page layout and interface
- `style.css` — dark theme styling
- `script.js` — conversion logic
- `project.json` — metadata for the OpenStudio collection

## Notes

- No build step required
- Open `index.html` directly in the browser
- Built with vanilla HTML, CSS, and JavaScript
