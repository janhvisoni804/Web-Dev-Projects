# Anti To-Do List

A productivity app that flips the idea of a to-do list.  
Instead of listing tasks you completed, you record distractions you successfully avoided.  
Build streaks for resisting time-wasters and celebrate your focus.

## Features
- Add avoided distractions (e.g., "Skipped scrolling Instagram").
- Daily streak counter for focus days.
- Stores streaks in localStorage.
- Clean dark UI with gradient buttons.
