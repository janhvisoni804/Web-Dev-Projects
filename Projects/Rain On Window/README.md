# Rain On Window

A standalone realistic rainy glass simulation built with HTML, CSS, Canvas, and vanilla JavaScript. The scene creates moving rain droplets on a blurred window surface with optional lightning flashes, synthesized thunder, and ambient rain audio.

## Features

- Real-time canvas rain with sliding droplets and fast background streaks
- Adjustable rain intensity from drizzle to heavy downpour
- Lightning toggle with randomized storm flashes
- Thunder toggle using synthesized Web Audio rumbles
- Ambient rain sound toggle with volume control
- Manual storm burst button for instant heavy rain and lightning
- Responsive layout for desktop and mobile screens

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- Web Audio API

## Usage

Open `index.html` in a modern browser. Use the storm controls to change intensity, enable lightning, turn on ambient rain, enable thunder, and trigger a short storm burst.

Browser audio starts only after the user enables a sound control or presses the storm burst button, matching modern autoplay rules.
