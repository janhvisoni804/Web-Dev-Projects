# Movie Search App

A sleek, modern, and highly efficient Movie Search App built with vanilla HTML5, CSS3, and JavaScript (ES6+). Search for movies, series, and episodes using the OMDb API and view detailed information in an elegant modal.

## Features

- Instant movie search by title
- Responsive grid layout with beautiful glassmorphism cards
- Detailed movie modal with poster, genre, director, actors, plot, and IMDb rating
- Keyboard accessible (Enter to search, Escape to close modal)
- Loading spinners and clear error/empty states
- Fully responsive from mobile to desktop

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern Grid/Flexbox layouts, custom properties, glassmorphism effects
- **Vanilla JavaScript (ES6+)** - Fetch API, async/await, event delegation

## Getting Started

Simply open `index.html` in any modern browser. No build step or dependencies required.

```bash
# If you have Python installed, you can run a quick local server:
python -m http.server 8000
```

Then navigate to `http://localhost:8000` in your browser.

## API

This project uses the free [OMDb API](https://www.omdbapi.com/) to fetch movie data.
