# YouTube Homepage Clone

A responsive YouTube homepage clone featuring:
- Sidebar navigation (Home, Trending, Subscriptions, Library)
- Responsive video grid
- Search bar with clear button
- Dynamic category filters (All, Music, Gaming, News)
- Font Awesome icons for polished UI

## How to Run
1. Clone the repository
2. Open `index.html` in your browser or run with Live Server
3. Test search, filters, and responsive layout
