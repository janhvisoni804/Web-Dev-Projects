# Password Generator

A simple and responsive Password Generator built with HTML, CSS, and JavaScript.

## Features

* Generate secure random passwords
* Choose password length (6–30 characters)
* Include:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Symbols
* Copy password to clipboard with one click

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Use

1. Adjust the password length using the slider.
2. Select the character types you want.
3. Click **Generate Password**.
4. Click the **Copy** button to copy the password.
