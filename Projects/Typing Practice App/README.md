# Typing Practice App

An interactive, premium web application built from scratch to help users analyze, practice, and scale their keyboard typing speed and typing accuracy.

## Features

- **Modern Glassmorphic Design**: Clean UI with immersive animated presets (Matrix Green, Cyberpunk Amber, Neon Purple) and micro-interactions.
- **Dynamic Word Arenas**: Adjustable test intervals (15s, 30s, 60s) and difficulty tiers (Easy, Medium, Hard).
- **Comprehensive Real-Time Analytics**: Calculate WPM (Words Per Minute), characters typed, accuracy percentage, and visual timeline charting representing speed velocity.
- **Virtual Interactive Keyboard**: Key mappings that light up dynamically on matching keyboard keystroke triggers.
- **Web Audio Engine Sounds**: Immersive clicking synthesizer audio triggers matching key interactions without static media dependencies.
- **History Dashboard**: LocalStorage caching preserving historical high scores, average metrics, and records.

## Tech Stack

- **Structure**: Semantic HTML5 markup
- **Styling**: Modern CSS3 (CSS Custom Variables, Flexbox, CSS Grid, Transitions)
- **Engine**: Vanilla Object-Oriented JavaScript
- **SFX Engine**: Web Audio API Sound Synthesizer

## Local Setup

### Direct Browser Access
No setup or build tooling required. Simply open `index.html` directly in your browser.

### Local Serving
Alternatively, you can run a local HTTP server inside the project directory:
```bash
npx http-server
```
