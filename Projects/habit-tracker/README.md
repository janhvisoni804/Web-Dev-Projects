# 🌱 Habit Bloom - Habit Tracker

A modern habit tracking application built with HTML, CSS, and JavaScript.

Habit Bloom helps users build consistency by tracking daily habits, monitoring streaks, and visualizing progress through a clean and responsive interface.

## Features

* Add and delete habits
* Mark habits as completed for the day
* Automatic streak calculation
* Daily progress tracking
* Local Storage support
* Dark/Light mode
* Responsive design
* Modern glassmorphism UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## Project Preview

Track habits, build streaks, and stay consistent with an intuitive and visually appealing dashboard.
