# Travel Booking Dashboard

An advanced, three-column responsive management dashboard designed to emulate corporate booking suites. Handles dual-mode selection sets and evaluates contextual aggregations instantly on user inputs.

## Key Features

- **Dual Tracking Modules**: Toggle immediately between Flight routes or Hotel suite options inside a single layout context.
- **Dynamic Selection Grid**: Update parameters and preview filtered card records matching destination queries instantly.
- **Reactive Financial Models**: Calculates running base fares, standard percentage taxation brackets, and net total projections as selections update.

## Tech Architecture

- Semantic HTML layouts grid structures.
- Pure state tracking loops built with Vanilla JS DOM manipulation routines.
