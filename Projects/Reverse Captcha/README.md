# Reverse Captcha // Humanity Proving Portal

Reverse Captcha is a satirical, full-stack browser verification simulator built to fulfill the behavioral testing constraints specified in tracking item **Issue #185**. Simulating a scenario where web infrastructure treats linear logic as an explicit bot signature, the platform requires users to inject chaos, abstract definitions, and flawed hand-drawn patterns to bypass system guards. Built entirely client-side using pure vanilla modules.

---

## Technical Capabilities & Core Specifications

### 1. Retro-Brutalist Core Scaffolding (`style.css`)

- **Strict Real Estate Resolution Constraint:** Anchors all operational layout branches inside a focused viewport panel footprint (`max-width: 1200px`, `height: 760px`) to prevent visual component fragmentation on varied monitors.
- **Dual-Personality Theme Aesthetics:** Unifies stark, cold grid terminal lines (`#070a13`) accented with luminous green operational borders (`#22c55e`) with organic human canvas inputs.
- **Jitter-Proof Monospaced Typography Locks:** Analytical progress sliders, verification quotient tallies, logs, and prompt arrays implement explicit monospaced font declarations (`ui-monospace, Consolas`), guaranteeing structural coordinates stay perfectly locked during numeric updates.
- **Visual State Shaders:**
  - `.terminal-card.failure-shake`: Fires an intense horizontal keyframe vibration path (`@keyframes crash-vibrate`) upon detecting rigid bot behaviors.
  - `.terminal-card.success-pulse`: Sweeps a deep, rich translucent neon backlight across layout hulls upon successful validation milestones.

### 2. Chaotic Pattern & Variance Testing Algorithms (`script.js`)

- **State Machine Verification Progression:** Governs progression parameters through a clean modular routing loop, checking state transitions dynamically to transition between multiple-choice, canvas sketch, and semantic descriptive modules.
- **Geometric Imperfection Canvas Parser:** Evaluates hand-drawn loops inside the playground module. Rather than looking for bounding-box collisions, the engine parses captured path stacks to isolate the centroid. It calculates continuous radial lengths ($r_i$) to target nodes across the track:
  $$\text{Centroid } (X_c, Y_c) = \left( \frac{1}{N}\sum X_i, \frac{1}{N}\sum Y_i \right)$$
  The system computes the absolute standard deviation variance across lengths. Perfect mathematical arcs containing negligible standard deviations ($\sigma \le 2.5\%$) are automatically flagged as artificial scripts, whereas organic configurations within target variances ($8\% \le \sigma \le 35\%$) pass cleanly.
- **Abstract Text Semantic Validation:** Evaluates loose text description blocks targeting open-ended strings. The parser screens characters for minimum length scales, filters strings against lists of literal machine code symbols (`{`, `}`, `[]`, `+=`), and verifies the inclusion of qualitative linguistic triggers to confirm human authorship.
