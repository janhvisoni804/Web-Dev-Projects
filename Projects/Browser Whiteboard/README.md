# Browser Whiteboard

A lightweight, interactive drawing board built with HTML5 Canvas, CSS, and Vanilla JavaScript. 

## Features
- **Customizable Brush:** Change colors and stroke thickness dynamically.
- **Light & Dark Boards:** Toggle the background canvas color between light and dark modes.
- **Canvas Clearing:** Easily wipe the board clean to start over.
- **Export Artwork:** Download your creations directly as a PNG image.

## Run It
Open `index.html` in any modern browser to start drawing. No server or build tools are required.
