# Number Connect Puzzle

A fun puzzle game where players connect matching numbers with paths.  
Paths cannot overlap and must fill the entire grid.

## Features
- Interactive 5x5 grid.
- Click to start a path from a number.
- Extend paths by clicking empty cells.
- Reset button to clear paths.
- Modern dark UI with gradient paths.

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
3. Play the puzzle by connecting numbers.
