# 💻 Coding Quiz Game

## Description

Added a Coding Quiz Game that allows users to test their programming knowledge through multiple-choice questions across various computer science topics.

## Features

* Multiple-choice programming questions
* Categories including Java, Python, JavaScript, DSA, and DBMS
* Real-time score tracking
* Quiz result summary
* Incorrect answer review system
* Category-based question selection

## Benefits

* Reinforces programming concepts
* Helps identify knowledge gaps
* Provides an interactive learning experience
* Supports exam and interview preparation

## Future Improvements

* Difficulty levels
* Timed quiz mode
* Leaderboard system
* Achievement badges
* Randomized question generation
* Progress analytics dashboard
