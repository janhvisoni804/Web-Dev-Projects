# Productivity Command Center

A modern, comprehensive, and interactive centralized productivity workspace. Designed for students, developers, and professionals, this application groups crucial productivity widgets—Task Management, Goal Tracker, Rich Note-taking, Habits Logger, Achievements Hub, and a Focus Center with custom Canvas Analytics—into one elegant dashboard.

## 🚀 Key Features

*   **centralized Dashboard**: At-a-glance view of daily stats, upcoming tasks, active habit streaks, Pomodoro controls, and motivational quotes.
*   **Task Management System (Kanban / List)**: Full CRUD capabilities, drag-and-drop simulated board, custom prioritization, due date alerts, status filters, and search.
*   **Goal Tracking Dashboard**: Set daily, weekly, or monthly goals, monitor milestones, and view interactive progress percentages.
*   **Notes Management Workspace**: Rich text editor representation with markdown notes, categorizations, search keywords, and color coding.
*   **Focus Center (Pomodoro Timer)**: Adjustable sessions/breaks, live tick-tock sounds, alert chimes, background status tracking, and focus history log.
*   **Consistency Habits Grid**: Interactive calendar grid tracking progress streaks, habit completion logs, and milestone rewards.
*   **Interactive Analytics Dashboard**: Beautiful charts rendered directly on HTML5 Canvas representing task completion ratios, daily/weekly scores, and categories distribution.
*   **Achievements & Milestones**: Unlocked badges based on user consistency, completed tasks, goals met, and hours focused.
*   **Keyboard Shortcuts**: Fast-access hotkeys to navigate sections and trigger controls instantly.
*   **Theme Engine**: Vibrant light and glassmorphic dark mode styles with smooth transitions.
*   **Full Data Import/Export**: Backup and restore all workspace information as a single JSON file. Offline-first design with `localStorage` sync.

## ⌨️ Keyboard Shortcuts

*   `Alt + D`: Navigate to **Dashboard**
*   `Alt + T`: Navigate to **Tasks**
*   `Alt + G`: Navigate to **Goals**
*   `Alt + N`: Navigate to **Notes**
*   `Alt + F`: Navigate to **Focus Center**
*   `Alt + H`: Navigate to **Habits**
*   `Alt + A`: Navigate to **Achievements**
*   `Alt + P`: Start / Pause Pomodoro Timer
*   `Alt + L`: Toggle Light / Dark Mode

## 🛠️ Technology Stack

*   **Structure**: Semantic HTML5 markup
*   **Styling**: Modern CSS3 using Flexbox, CSS Grid, Custom Variables (Themes), HSL Color Palette, and glassmorphism.
*   **Scripting**: Pure Vanilla JS (ES6+) utilizing Canvas API, LocalStorage API, Audio Synthesis, and key event registers.

## 🚀 How to Run

1. Clone or download the repository.
2. Locate the folder `Projects/Productivity Command Center/`.
3. Double-click `index.html` to launch the application directly in any modern browser. No setup or build processes are required.
