# UI Color Palette Extractor

Extract dominant color palettes from any image. Upload a JPG, PNG, or WEBP image and instantly get a clean set of representative colors with hex and RGB values.

## Features

- Drag-and-drop or click to upload
- Extract 5, 8, 12, or 16 dominant colors
- Click any swatch to copy its hex code
- Export palette as CSS custom properties

## Usage

1. Open `index.html` in any browser
2. Drop an image onto the upload zone or click to browse
3. Select the number of colors from the dropdown
4. Click swatches to copy hex values or click "Export CSS"

## File Structure

- `index.html` — Application shell
- `style.css` — Clean, minimal styling
- `script.js` — Image processing and palette extraction
- `README.md` — This file
- `project.json` — Project metadata
- `thumbnail.svg` — Preview graphic
