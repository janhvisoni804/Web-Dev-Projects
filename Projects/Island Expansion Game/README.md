# Island Expansion Game

A puzzle game built with vanilla HTML, CSS, and JavaScript. Assign Land (green) and Ocean (blue) tiles to match the numbered island root cells. Each island must grow to exactly match its number.

## Features
- **3 levels**: Tropical 5×5, Archipelago 7×7, Continent 10×10
- **Flood-fill validation**: Recursive check ensures each island matches its target size
- **Error highlighting**: Wrong tiles flash with shake animation
- **Level persistence**: Current level saved to localStorage
- **Oceanic dark theme**: Deep navy backdrop with emerald land and sky-blue ocean
