# Invoice & Billing Generator

A professional web app to **create invoices, add line items, calculate totals, and export as PDF**.  
Built with **HTML, CSS, and JavaScript** for the [Web-Dev-Projects](../README.md) showcase.

## 🚀 Features
- Add client details (name, invoice number, date)
- Dynamic line items (description, quantity, unit price)
- Auto-calculated subtotal, tax, and grand total
- Export invoice as PDF using [jsPDF](https://github.com/parallax/jsPDF)

## 📂 Project Structure
Invoice-Billing-Generator/
│── index.html      # Main UI
│── style.css       # Styling
│── script.js       # Logic & PDF export
│── README.md       # Documentation



## 🛠️ How to Use
1. Open `index.html` in your browser.
2. Fill in client details and add line items.
3. Totals update automatically.
4. Click **Export as PDF** to download your invoice.
