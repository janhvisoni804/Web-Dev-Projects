# Developer Portfolio Builder

A premium developer portfolio builder with live preview, customizable sections, and responsive export-ready layouts.

## Features

- **Dual Templates**: Choose between Classic Minimalist and Modern Professional designs
- **Real-time Preview**: See changes instantly as you fill in your information
- **Customizable Colors**: 6 accent color options to personalize your resume
- **Photo Upload**: Add a profile photo using the File Reader API
- **Dynamic Sections**: Add multiple entries for experience and education
- **Skills & Languages**: Tag-based system for adding and managing skills
- **Print to PDF**: Generate a professional PDF with a single click
- **Local Storage**: Your resume data is automatically saved to your browser
- **Responsive Design**: Works seamlessly on all devices

## How to Use

1. **Fill in Personal Info**: Enter your name, title, email, phone, website, and professional summary
2. **Add Experience**: Click "+ Add" to add work experiences with company, role, dates, and descriptions
3. **Add Education**: Add your educational background with schools, degrees, and dates
4. **Add Skills**: Type skills one at a time and press Enter to add them as tags
5. **Add Languages**: Similarly add languages you speak
6. **Upload Photo**: Click the photo upload area to add a profile picture
7. **Choose Template**: Switch between Classic and Modern templates from the toolbar
8. **Select Colors**: Pick your preferred accent color from the color swatches
9. **Download**: Click "Download PDF" to save your resume
