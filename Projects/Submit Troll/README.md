# 🕹️ Submit Troll

A playful interactive project where the **"Yes" button constantly escapes your cursor**. Try to catch it if you can! Perfect for fun demos, troll projects, and showcasing interactive UI tricks.

---

## ✨ Features
- 🎯 Escaping "Yes" button that dodges your mouse
- ⚡ Difficulty levels (Easy, Medium, Hard)
- 🎭 Fake catches (button pretends to be caught but escapes)
- 📊 Statistics tracking (attempts, catches, escapes)
- 📱 Mobile support (touch events)
